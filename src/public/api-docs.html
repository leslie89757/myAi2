ChatGPT API 文档

版本: 1.0.0

此 API 提供了与 ChatGPT 交互的能力，包括流式和非流式两种接口。

API 端点

POST /api/simple-chat

非流式聊天接口，一次性返回完整的 ChatGPT 回复。

请求参数

参数名 类型 必填 描述
message string 是 用户发送的消息内容

请求示例

{
  "message": "你好，请介绍一下自己"
}

响应示例

{
  "reply": "你好！我是 ChatGPT，一个由 OpenAI 训练的大型语言模型。我可以回答问题、提供信息、进行对话，以及帮助完成各种文本相关的任务。我的知识库截止到我的训练数据结束的时间点，所以关于之后发生的事件我可能无法提供信息。有什么我可以帮助你的吗？"
}
POST /api/stream-chat

流式聊天接口，使用 Server-Sent Events (SSE) 实时返回 ChatGPT 回复。

请求参数

参数名 类型 必填 描述
message string 是 用户发送的消息内容

请求示例

{
  "message": "你好，请介绍一下自己"
}

响应格式

响应以 SSE 格式返回，每个事件包含一个 JSON 对象，其中 content 字段包含回复的一部分文本。最后一个事件为 [DONE] 表示流结束。

响应示例

data: {"content":"你好"}

data: {"content":"！我是"}

data: {"content":" ChatGPT"}

data: {"content":"，一个由"}

data: {"content":" OpenAI"}

data: {"content":" 训练的大型语言模型。"}

data: [DONE]

测试页面

本 API 提供了两个测试页面，方便您直接在浏览器中测试 API 功能：

  • 简单聊天测试页面 - 测试非流式接口
  • 流式聊天测试页面 - 测试流式接口（带打字机效果）

健康检查

GET /health

检查 API 服务是否正常运行。

响应示例

{
  "status": "ok"
}
